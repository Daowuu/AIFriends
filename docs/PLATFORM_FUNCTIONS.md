## 1. 当前产品定位

AIFriends 当前是一个角色驱动的 AI 聊天产品原型，重点是：

- 角色一致性
- 关系连续性
- 语音一致性
- 创作者可控
- 运行时可诊断

平台当前分成两条主要路径：

1. 用户路径  
   发现角色、添加好友、正式聊天。
2. 创作者路径  
   在 Studio 中配置角色、试听、试聊、查看诊断。

## 2. 当前核心页面

### `/`

首页内容流。

当前职责：

- 浏览公开角色
- 搜索角色
- 进入角色正式聊天页
- 进入作者空间页

### `/chat/:characterId`

正式聊天页。

当前职责：

- 与角色进行文字聊天
- 使用语音输入
- 接收角色语音播报
- 重置聊天窗口
- 保持面向终端用户的纯聊天体验

聊天页保持面向终端用户的纯聊天体验，角色配置和工程态诊断集中在 Studio。

### `/studio`

创作者主入口。

当前职责：

- 选择角色或新建角色草稿
- 编辑角色公开信息
- 编辑 `custom_prompt`
- 配置 AI 行为参数
- 配置音色
- 试听音色
- 试聊角色
- 查看运行时摘要和最近调试信息

这是当前角色工作流的主页面。  
`/workspace` 仅作为兼容入口，会重定向到这里。

### `/friends`

好友 / 关系页。

当前职责：

- 查看已建立关系的角色
- 移除好友
- 继续进入正式聊天页

### `/profile`

个人资料页。

当前职责：

- 编辑用户名、展示名、简介、头像
- 进入用户级 runtime 设置页

### `/settings/api`

用户级 runtime 设置页。

当前职责：

- 启用或关闭个人聊天模型配置
- 配置用户自己的聊天 runtime
- 配置独立 ASR runtime
- 查看当前 runtime summary

注意：

- 这个页面仍然存在
- 但对创作者来说，更推荐从 `/studio` 进入工作流

### `/space/:id`

用户空间页。

当前职责：

- 查看某个用户的公开资料
- 查看其公开角色
- 如果是本人空间，可继续编辑自己的角色

## 3. 当前角色工作流

当前推荐的创作者工作流：

1. 进入 `/studio`
2. 创建角色草稿或切换到已有角色
3. 配置角色公开信息
4. 配置 `custom_prompt`
5. 配置 AI 行为参数
6. 配置音色
7. 在 `试聊诊断` 面板中试听和试聊
8. 确认效果后，再从 `/chat/:characterId` 进入正式聊天

## 4. 当前 AI 相关产品能力

平台当前已经暴露给产品层的 AI 能力包括：

- 结构化 prompt 分层
- 角色自定义 prompt
- 轻量长期记忆
- 语音输入
- 语音播报
- 试听与试聊
- runtime summary
- 最近一次 debug 摘要

平台当前的主线能力集中在角色聊天、记忆、语音和诊断。
